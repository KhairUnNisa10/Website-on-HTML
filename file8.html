HYPER TEXT MARKUP LANGUAGE
(HTML)

An HTML is not a Programming Language! New Web developers may mistake HTML for a programming language when it is actually a markup language. HTML is used with other technologies because all HTML really does is organize documents. On the client side, JavaScript (JS) is used to provide interactivity. On the server side, a Web development platform like Ruby, PHP or ASP.NET is used.

WHAT IS HTML?

HTML stands for Hyper Text Markup Language. It is one of the most famous and widely used markup language, especially for web development. HTML act as the founding stone of any web page along with other web technologies like CSS, JavaScript. One can not neglect importance of HTML. There are uncountable uses of HTML.
One of the main use of HTML is to structure the web pages. It is very easy and very useful computer language. Probably Most of the software engineer from any domain started their computer career from coding in HTML.

HTML IN TECHNICAL DETAIL

Developers divides web development into two main classes:

  • Front-End
  • Back-End

More frankly HTML is considered as front-end technology. HTML language consists of the number of tags each tag has the opening tag and ending tag except some exceptional cases.
Opening tag starts like <open> this and ends like </close> this.

USES OF HTML

HTML is the most important, common and easy to use computer language. It belongs to markup language category. HTML is to web development, what water is to life. Uses and importance of HTML is abundant. HTML occupies the big number of internet’s volume. It is the foundation to the web development. Without HTML your browser cannot be able to display you a website.

  • Structuring the digital document
  • Developing web pages
  • Development of websites
  • Creating digital front-end

SYNTAX:

The syntax of HTML is very simple to understand. Every content is represented by a tag. 

 
	<html>
		<body>
			<p> <strong> This is a paragraph </strong> </p>
		</body>
	</html>

OUTPUT:

This is a paragraph

<p> tag defines text as a paragraph and <strong> tag make the text bold.

SUMMARY OF HTML TAGS

Summary Of HTML Tags
HTML TAGS
Tag Description Semantic/Non-Semantic
<title> </title> The <title> tag is used for the title of a webpage. Semantic!
<p> </p> The <p> tag is used for paragraph. Each and every paragraph must be written in <p>tag.
<center> </center> The <center> tag is used to center align the text but it is done through the CSS. Non-Semantic!
Definition Tags:
  • <dd> </dd>
  • <dt> </dt>
  • <dl> </dl>
 The <dd>, <dt> and <dl> tags are formed to make a complete definition with title and description. Semantic!
<mark> </mark> <mark> tag is used for highlighting a specific content.
<table> </table> The <table> tag is used to define a table within the html. In this tag the <tr>, <th>, and <td> tags are used to make it a complete table.

HOW HTML WORKS IN BROWSER.

HTML pages stores over the server on the internet. Whenever a user requests a web page, the browser generates a HTTP request and forward it to the server. The server responds to request and forward requested web page to browser back in HTML format. The browser then parse the HTML code into the human-readable form. The important thing to remember here is that browsers are parser of HTML.
