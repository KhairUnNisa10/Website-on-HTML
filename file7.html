JAVASCRIPT LANGUAGE

JavaScript often written as JS. It is a powerful and very popular high-level programming language (also known as the scripting language). Commonly used in writing web applications. Introduction of JavaScript is quite a long topic but we, summarize it for you.
It is one of the core and fundamental language of the web development. It is also part of the triads of the web. This language is best for describes the behavior of web pages and adds interactive nature to websites. JS is commonly being used in web development, but can also be used in developing powerful desktop applications.

HISTORY OF JAVASCRIPT

The developers of Netscape browser wanted to develop such language which has the ability to increase the functionality of web pages. This idea results in the development of JavaScript. However, They named it Mocha in May 1995, renamed it to LiveScript in Sep 1995, and then finally they again renamed it to JavaScript in Dec 1995. After renaming it to JavaScript they submitted their work to ECMA (an international organization for standardization). In June 1997 first official release of JavaScript is made under ECMA with code name ECMA-262. Currently, the latest stable version of JavaScript is ECMAScript 2017.

JavaScript IS NOT JAVA

Many people get confused about Java and JavaScript. They consider them as two different names pointing to java programming language, but that’s not true, both are different languages with a different name and different functionality. Although Netscape adopts some ideas from Java, but both are for entirely different purposes. Java is WORA (Write once, run anywhere) based powerful programming language dominating programming world since the 90s. On the other hand, JavaScript is a popular language mainly used for creating a dynamic and interactive website. It is famous as a client-side language.

In short, JavaScript is not java both have different purposes, syntax, and design.

WRITING STYLE

The writing style of JavaScript is camel case or camelCase, It is a technique of writing text in which word in the middle of the phrase is in capital format without punctuation and spaces.
Prerequisites: HTML , CSS

WHERE TO WRITE JavaScript IN HTML TAGS

You can write JavaScript anywhere in a web page. It is up to you. Some writes code in a separate file, some writes within a page, some writes before and after the body tag,others write in between the body. You can adopt any method you want.
JavaScript Tip: Putting <script> tag at the end of the document but before the closing body will load your page faster. This is because the browser will load HTML and CSS before JavaScript. Like below:

<html>
<head> </head>
<body>
<h1> JavaScript </h1>
<p> this a parapraph </p>

<script> document.write('JavaScript'); </script>

</body>
</html>
